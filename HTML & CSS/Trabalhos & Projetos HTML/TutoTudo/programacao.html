Tutoriais de Programação

Aprenda os fundamentos da programação e comece a criar seus próprios projetos com tutoriais simples e práticos.

HTML

HTML é a linguagem que estrutura as páginas da web. Com ela, você cria títulos, parágrafos, imagens, links e muito mais.

Exemplo básico de uma página HTML:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <title>Minha primeira página</title>
</head>
<body>
  <h1>Olá, mundo!</h1>
  <p>Este é meu primeiro site.</p>
</body>
</html>

Quer aprender mais? Acesse o tutorial completo de HTML no W3Schools.

CSS

CSS é a linguagem que estiliza as páginas HTML. Com ela, você define cores, fontes, tamanhos, layouts e efeitos visuais.

Exemplo básico para mudar a cor do texto:

h1 {
  color: blue;
}
p {
  font-size: 16px;
}

Quer aprender mais? Confira o tutorial completo de CSS no W3Schools.

Python

Python é uma linguagem de programação fácil e poderosa, usada para automação, análise de dados, web, entre outras áreas.

Exemplo simples que imprime uma mensagem:

print("Olá, mundo!")

Quer aprender mais? Veja a introdução oficial ao Python.
